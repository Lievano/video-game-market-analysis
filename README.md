# Video Game Market Analysis

Strategic market intelligence project for identifying video game success patterns across platforms, genres, regions, and review signals.

## Project Access

- English documentation: [README_EN.md](README_EN.md)
- Documentación en español: [README_ES.md](README_ES.md)
- Notebook EN: [notebooks/project_EN.ipynb](notebooks/project_EN.ipynb)
- Notebook ES: [notebooks/project_ES.ipynb](notebooks/project_ES.ipynb)

## Quick Start

```bash
pip install -r requirements.txt
jupyter notebook notebooks/project_EN.ipynb
```

## What This Project Demonstrates

- Market analysis for video game sales
- Platform lifecycle and product-generation dynamics
- Regional customer profile analysis
- Review-score correlation analysis
- Hypothesis testing with variance diagnostics
- Reusable analytical functions in `src/`

## Key Insight

Video game success is not explained by one variable. Commercial performance emerges from platform timing, genre-market fit, regional preferences, and review signals.
